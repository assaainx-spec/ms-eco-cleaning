[23-03-2026] BROWSER LANGUAGE AUTO-DETECTION
----------------------------------------------------------------
Added automatic language redirect on index.html:
  - On first visit, browser language (navigator.language) is read
  - Polish browser (pl-*) → redirected to pl.html
  - Non-Dutch browser → redirected to en.html
  - Dutch browser (nl-*) → stays on index.html
  - User preference saved to localStorage ('preferredLang')
  - Manual language switcher clicks update localStorage preference
  - Returning visitors go to their remembered language

All lang buttons (topbar + mobile drawer) across all 3 pages
now save the chosen language to localStorage on click.
